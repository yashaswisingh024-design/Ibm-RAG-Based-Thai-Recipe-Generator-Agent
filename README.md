# IBM-PROJECT
# 🍜 RAG-Based Thai Recipe Generator Agent

## Overview

The RAG-Based Thai Recipe Generator Agent is an AI-powered assistant that helps users retrieve authentic Thai recipes using Retrieval-Augmented Generation (RAG).

The system searches a secure recipe knowledge base and generates accurate responses using IBM Granite Models through IBM watsonx Orchestrate.

---

## Features

- 🍲 Thai recipe retrieval
- 🤖 AI-powered conversational assistant
- 📚 Retrieval-Augmented Generation (RAG)
- 📄 Knowledge Base using PDF cookbook
- 🍽️ Step-by-step cooking instructions
- 👨‍🍳 Chef assistance
- 📖 Recipe recommendation
